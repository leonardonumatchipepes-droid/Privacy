Política de Privacidade

Menongue Educa respeita a privacidade dos seus utilizadores.

1. Dados coletados: O aplicativo não coleta dados pessoais sensíveis.

2. Uso: Os dados são usados apenas para melhorar a experiência do utilizador.

3. Armazenamento: Conteúdos podem ser armazenados localmente no dispositivo.

4. Partilha: Não partilhamos informações com terceiros.

5. Contacto: Para dúvidas, contacte a equipa de desenvolvimento.
