Política de Privacidade

O aplicativo Menongue Educa respeita a privacidade dos seus usuários.

Coleta de Dados

Este aplicativo não coleta dados pessoais sensíveis dos usuários.

Uso das Informações

Os dados, quando necessários, são utilizados apenas para melhorar a experiência do usuário.

Armazenamento

O aplicativo pode armazenar conteúdos localmente no dispositivo para funcionamento offline.

Compartilhamento

Não compartilhamos informações com terceiros.

Contato

Para dúvidas, entre em contato com o desenvolvedor.
